# fold_line.nvim

lines, not for indent, but for fold.

## install

Using [lazy.nvim](https://github.com/folke/lazy.nvim)

```lua
-- init.lua:
{
    "gh-liu/fold_line.nvim",
    event = "VeryLazy",
}
```

## Appearance

Use the highlight group `FoldLine` to change the color of the line. Per default it links to `Folded`.
